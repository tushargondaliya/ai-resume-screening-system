# 🤖 AI Resume Screening System

An AI-powered system that automates resume screening and ranks candidates based on job requirements.

## 🚀 Features
- Resume Upload & Parsing
- AI Skill Extraction
- Job Matching Algorithm
- Candidate Ranking System
- Dashboard Analytics

## 🛠 Tech Stack
- Python Flask
- HTML, CSS, JavaScript
- SQLite
- NLP Algorithms

## ▶️ Run Project
pip install -r requirements.txt
python app.py
